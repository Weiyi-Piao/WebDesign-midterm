What is inheritance?

When you are creating a class in whatever programming language you choose, there is a delimma:
The two classes that you created are mostly the same (for example: 90% of the program of one class
is just a simply copy of another class.) In this case, what would you do?

You may choose one of the follow technique:

  • REPEATEDLY writing the same code for several times.
  • OR Use the technuque called Inheritance

For the first solution, the program does worked as intended.
But what should you do if there are 10 similar classes? 20 class? or even more?

It seems too redundant for a program. that's why we should use Inheritance:
Inheritance create lowerclass that can use all the functions from the superclass.
